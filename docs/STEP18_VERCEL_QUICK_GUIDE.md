# Step 18 — Vercel Quick Guide

1. Create or open Vercel account.
2. Create a new project.
3. Import the GitHub repository or upload/connect the React project.
4. Set root directory to `omideno7-meetings-react` if repository contains multiple folders.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.
8. Open the HTTPS link on phone.
9. Install as PWA.

## Environment variables for test

For this visual test, Supabase and LiveKit can stay empty.

Later add:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_LIVEKIT_URL`
