# Step 18 — Phone Install Test Checklist

After the deploy link is ready:

## iPhone test

- Open link in Safari
- Confirm landing page appears
- Tap Share
- Tap Add to Home Screen
- Confirm icon and name
- Open installed app from Home Screen
- Test demo owner login
- Test demo member login
- Test Install App page
- Test offline page by turning internet off and reopening

## Android test

- Open link in Chrome
- Confirm install prompt or menu option
- Tap Install App / Add to Home screen
- Open installed app
- Test demo owner login
- Test demo member login

## Not production yet

Do not use this for real members yet because Supabase Auth and LiveKit are still not connected.
