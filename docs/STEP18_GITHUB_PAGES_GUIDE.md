# Step 18 — GitHub Pages Guide

This option uses:

```text
.github/workflows/deploy-github-pages.yml
```

Steps:

1. Put `omideno7-meetings-react` at repository root, or adjust workflow paths.
2. Push to `main`.
3. Go to GitHub repository settings.
4. Enable Pages from GitHub Actions.
5. Run workflow.
6. Open the generated Pages URL.

For the first test, Vercel or Netlify is usually easier.
