# Step 18 — Netlify Quick Guide

1. Create or open Netlify account.
2. Add new site.
3. Connect GitHub or deploy manually.
4. Set base directory to `omideno7-meetings-react` if needed.
5. Build command: `npm run build`.
6. Publish directory: `dist`.
7. Deploy.
8. Open the HTTPS link on phone.
9. Install as PWA.
